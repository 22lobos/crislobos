# Sistema de Gestión Financiera para Banco Internacional

## Descripción

Este proyecto es una aplicación en Python para un banco internacional que permite analizar y gestionar los datos financieros de sus clientes. Incluye funcionalidades como asignación de saldos aleatorios, clasificación de saldos, cálculo de estadísticas avanzadas, generación de reportes detallados y opción para salir del programa.

## Funcionalidades

- Asignación de saldos aleatorios a 10 clientes.
- Clasificación de saldos en tres rangos específicos.
- Cálculo de estadísticas avanzadas sobre los saldos.
- Generación de reportes detallados y exportación a archivos CSV.
- Opción para salir del programa con mensaje de despedida.
